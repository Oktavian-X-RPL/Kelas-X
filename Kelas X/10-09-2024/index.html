Belajar HTML Dan CSS

HTML Digunakan untuk membuat tampilan website

CSS digunakan untuk mengatur tampilan

Materi HTML

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untuk tabel
  • input untuk form
  • li untuk list atau data

Materi CSS

  1. margin, padding, position, untuk mengatur jarak antara elemen HTML
  2. display untuk mengatur elemen HTML agar tampil atau tidak tampil, mendatar/menurun
  3. style untuk mengatur font
